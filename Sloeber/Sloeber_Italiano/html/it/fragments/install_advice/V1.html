IDE Arduino versione 1¶

La V1 del plugin funziona con Juno e Indigo. Non so se funzionino Kepler o Luna.

Non esiste alcun prodotto nella V1.

La V1 è considerata «completata» e non ci aspettiamo nuove modifiche.
